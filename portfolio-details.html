Service Buyer Mobile Application

React Native Product
March 09, 2026
  • Android and iOS
  • React Native, Firebase
  • Buyer and provider roles

Service Buyer is a two-sided marketplace app where users can discover trusted professionals, request services instantly, and track progress in a clean and simple workflow.

The focus of this build was reducing user friction. I designed a direct booking flow, fast sign-in via OTP, and lightweight profile screens that keep key actions visible without visual clutter.

Core Features

  • Real-time service request and order status updates.
  • OTP authentication with secure token refresh flow.
  • In-app payment integration with transaction tracking.
  • Dual dashboard experience for buyer and provider accounts.
